## 寂寞飲者《紅樓夢》工作室。

### 寂寞飲者也評《石頭記》 

#### 《紅樓夢》版本比較、解註、聽書、研究平臺。

愚，自稱「寂寞飲者」厚顔效仿脂硯齋等批書者，注上敝人對《紅樓夢》的見解。
批《紅樓夢》者非止脂硯一人，還有綺園，畸笏叟，松齋等署名，所以加上我一個並不爲過。
雖脂批對了解《紅樓夢》大有助益，然批語有合理的，亦有少數不合理的，更有不知所云之處。
對難解的、或值得推敲的，我亦稍加注解，發表拙見。

我非《紅樓夢》學者，亦非文學專業，只因喜歡《紅樓夢》，見網間版本雑多而缺乏整理校誤，
又有光學OCR掃描文字辨認、簡繁轉換之錯誤，故不自量力，閑暇之餘，以下列資源爲藍本，
將其加以整理校正，並建構一個可互動的網上閲讀、聽書、研究平臺，稱作「寂寞飲者也評《石頭記》」。

    維基文庫的《脂硯齋重評石頭記》[https://zh.wikisource.org/wiki/脂硯齋重評石頭記]
    台大中文系 徐富昌 博士整理的《紅樓夢•精緻乙版》[附注：原始網頁已不復存在]
    《癸酉本石頭記》後二十八回 [附注：原始簡體版網頁已不復存在
    目前可查内容如後：https://vicalloy.github.io/the-guiyou-version-of-dream-of-the-red-chamber]

【主旨】：之所以整理這個校注本，是想要論證《癸酉本》後二十八回就是《紅樓夢》所佚失之舊稿。
一、脂硯齋等批語提供甚多支持與證據。脂硯等批語提到八十回後的情節，
    對證並增加《癸酉本》後二十八回之内容的可信度。
二、雖《程高本》後四十回是依據風雨侵蝕、模糊不可辨的「漶漫舊稿」截短補長而成， 
    然仔細比較《程高》後四十 與《癸酉》後二十八，讀者可以發現《程高》與《癸酉》
    有許多類似、甚至一字不差的文字重疉。因此我特別的將《程高》後四十回 與《癸酉》後二十八回，
    相同類似的文字，一一指出，相互印證。

這些不一而多的綫索可用來論證《癸酉》後二十八應該就是脂硯齋與畸笏叟時常「嘆嘆」的
《紅樓夢》佚失之舊稿。網路上對《癸酉本》的詬病多是文字粗鄙，然《癸酉本》應該是未經曹雪芹
仔細增刪校閲的一個版本。《癸酉本》後二十八與《紅樓夢》前八十的情節， 絲絲入扣，前後呼應，若合節符。
我在注解和評語中將其一一指出。請各位指教。

古人作詩寫文，講究「起承轉合」、「對仗工整」。如將《癸酉本》後二十八與《紅樓夢》前八十合並來看， 
整部書之「起承轉合」、「對仗工整」明顯可見，堪稱完美。《紅樓夢》第一回，從青埂峰石頭意欲遊歷人間而「起」，
剛好太虛幻境一干風流冤孽也想造劫歷世。接著，霍啓（禍起）丟失英蓮，黛玉拋父進京，「承」接了故事的開始。
整部書一百零八回，高潮恰恰出現在第五十四回之末，用鳳姐的「咱們也該『聾子放炮仗 ── 散了』罷」，雙關語，
來暗示賈府由盛「轉」衰的拐點。故事情節極其工整地鋪陳排序、對仗開來，來龍去脈，前後對應，
絲絲相扣，「處處交代」。幸喜的是，遺失的内容清清楚楚的、相互對應的、出現在《癸酉》後二十八之情節中，
正如脂硯常説的「一點不亂」。最後收尾在《癸酉本》第一百零八回，警幻仙姑發佈《情榜》，
石頭回歸青埂峰，一切歸於平靜而止，此爲「合」也！

令人拍案的是，原來《紅樓夢》的結局早在甄士隱注解《好了歌》時，已在第一回裡寫出。
完全符合作者喜歡用晦澀文字提前「劇透」的寫作風格。曹雪芹素喜「劇透」、並打臉書中人物。
行文時，常先有「説嘴」之句，後就有「打嘴」之文。而《癸酉》後二十八亦顯示這些文字筆法、寫作邏輯。
學者不該被所謂的「文字粗鄙」之説而忽略了藏在《癸酉本》字裡行間的種種許多細微證據。 
再論《程高》與《癸酉》的重疉：雖然中舉之人，兩個版本沒能完全對上，但最蹊蹺的是，中舉的名次，
兩個本居然都是「第七名」與「第一百三十名」。會試取中者一般二三百名。若是不同的底稿，
兩版的作者在兩百名中任意取兩個數字，結果取出相同組合的兩個數字，是 15億分之一的機率。
若論「對仗」，略舉幾例，以供參考：

    揚州城外，雨村巧遇冷子興 ──對應── 雨村審薛蟠，再遇冷子興。
    葫蘆僧門子提醒雨村「賈王史薛」乃護官符也 ──對應── 冷子興提醒雨村「賈王史薛」已敗不足懼。
    薄命司圖冊判詞 ──對應── 《癸酉》後二十八回中，各人物之結局。
    晴雯、黛玉的結局 ──對應── 「一個是水中月（的晴雯），一個是鏡中花（的黛玉）」，皆與寶玉「無緣」而逝。
    襲人、寶釵的結局 ──對應── 「一個枉自嗟呀（的襲人），一個空勞牽掛（的寶釵）」，皆被寶玉「破瓜」而棄。
    閙學堂，寶玉逼金榮道歉，「早知日後閒爭氣」之句 ──對應── 金榮新朝爲官，街道上痛打寶玉。
    因麒麟伏白首雙星 ──對應── 最後白首陪寶玉的正是湘雲。
    黛玉記下寶玉説了兩次去當「和尚」的話 ──對應── 寶玉後來真的做了兩次和尚。
    雨村路過湘江，見一老嫗倒在雪中，「正嘆他人命不長」 ──對應── 「那知自己歸來喪」，
    不幸發現，被流放的葫蘆僧門子已升高官，要對雨村打擊報復。

吳新雷、黃進德 在《曹雪芹江南家世考》裡，指其祖曹寅任江南織造時，受有康熙密旨，交遊江南文人，並對其安撫。
我猜，或因從小受到江南文人的耳濡目染，和被雍正抄家的緣故，曹雪芹對雍正的微詞，隱藏在《石頭記》的寫作之中。
《癸酉本》後二十八回内容有著對朝政的評論，甚至「驅逐韃虜」的字眼，這些内容或許是曹雪芹遲遲未能增刪修改後二十八回的原因。
所以《第一回》裡，當提到《石頭記》「毫不干涉時事」時，脂硯齋趕忙强調說：「要緊句。」真是「此地無銀三百兩」！
或許因好友希望先睹爲快，借閲不還，致使後二十八回之内容淪落紅塵、塵封無聞至今。
雖然脂硯齋等批書者時時抱憾遺失之内容，但這也許是上天的眷顧，讓大家因此躲過了一場可能的文字獄。
石頭版本何其之多？我猜，亦如雨村，雪芹也是「賣文爲生」。邊寫邊改，邊抄邊賣。
數年積纍下多種「大同小異」的版本，而《癸酉本》則一早期的版本也。我的猜測到此爲止。

這個《紅樓夢》閲讀、聽書、研究平臺，特意將《癸酉本》後二十八回續接於《脂硯齋重評石頭記》前八十回之後。
並將脂硯齋等批書者之批語，或涉及、或對應《癸酉本》後二十八之内容處加以注解。爲了避免混淆，
將它標記作《脂硯重評并癸酉後二十八回校注本》，或簡稱《寂寞飲者校注本》。此閲讀平臺仿照西洋《聖經》，
試著將《紅樓夢》分節、編號，以方便未來學者與讀者索引、翻查、對比。並將網上的異體字拼凑法，比如(左金右夅)者，
改作相對應的正體「𨦟」，並加以注解（如，「鋒」之異體）。若無相對應的正體字數位字體，則用圖像的方式來代替字串拼凑。
本平臺也提供語音説書：以有聲讀物之方式，推廣閲讀。更注解了一些生澀的詞彙，
期能方便年輕讀者理解書中包羅萬象的内容、種種人情世故、文化細節的隱示、等等。
《紅樓夢》實乃中華文化之瑰寶，曹雪芹受乃祖曹寅的濡染，對戲曲有廣汎的知識。
建議讀者在閲讀《紅樓夢》時，把欣賞傳統戲曲的韻律、動作與想象帶入閲讀之中。

我自2019年起，這些年來的整理校注，頗能體會曹雪芹於悼紅軒中，披閱十載增刪五次的辛勞與快樂。

或相關的影音圖片。此乃敝人螳螂弱臂爲承先啓後少儘綿薄的努力。恕我手笨眼拙，若有謬誤還望大德指正。謝謝。
